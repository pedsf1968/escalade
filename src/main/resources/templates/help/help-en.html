Climbing community site project for a fictional association “Les amis de l'escalade”.

The purpose of this site will be to connect and share information. It allows to give visibility to the association in order to encourage independent climbers to join it.

Any user can see the Member Topographies.

A member can upload his Topographies

A member can reserve a Topography

Topography

A topography is a collection containing all the useful information on climbing sites in a region.

It is characterized by information, a photo, a map, a description

It can contain sectors and tracks.

It is managed by a manager

Sector

It is characterized by information, a photo, a map, a description

It may contain pathways.

Way

It is characterized by information, a photo, a map, a description

It can contain lengths.

Length

A length is a part to climb of a route. There is a relay at the top of each length.

It is made up of Spits anchor points.

Spit

Anchor point for the length that can be fitted.
