Proyecto de sitio comunitario de escalada para una asociación ficticia "Les amis de l'escalade".

El propósito de este sitio será conectarse y compartir información. Da visibilidad a la asociación para alentar a los escaladores independientes a unirse.

Cualquier usuario puede ver las Topografías de miembros.

Un miembro puede subir sus Topografías

Un miembro puede reservar una Topografía

Topografía

Una topografía es una colección que contiene toda la información útil sobre sitios de escalada en una región.

Se caracteriza por información, una foto, un mapa, una descripción

Puede contener sectores y pistas.

Es administrado por un administrador

Sector

Se caracteriza por información, una foto, un mapa, una descripción

Puede contener vías.

Camino

Se caracteriza por información, una foto, un mapa, una descripción

Puede contener longitudes.

Longitud

Una longitud es una parte para escalar una ruta. Hay un relé en la parte superior de cada longitud.

Está formado por puntos de anclaje de Spits.

Spit

Punto de anclaje para la longitud que se puede ajustar.
